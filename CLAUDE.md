# 本次任务内容

## 项目分析结果

经过全面审阅，发现这是一个为 Claude Code 构建专业提示词模板的项目，主要包含：

- **general/**: 通用模板和指南
- **rust/**: Rust 语言的各种项目类型模板
- **specialization-agents/**: 专门化的开发者子代理定义

项目目前已经有了 5 种 Rust 项目模板：CLI应用、Web服务、库项目、跨平台项目、高性能系统项目。

## 具体任务

根据 README.md 的要求，需要创建针对不同 Rust 应用类型的特化子代理：

1. **创建 CLI 应用开发子代理** - 基于 `CLAUDE_CLI应用模板.md`
2. **创建 Web 服务开发子代理** - 基于 `CLAUDE_Web服务模板.md` 
3. **创建库项目开发子代理** - 基于 `CLAUDE_库项目模板.md`
4. **创建跨平台项目开发子代理** - 基于 `CLAUDE_跨平台项目模板.md`
5. **创建高性能系统开发子代理** - 基于 `CLAUDE_高性能系统项目模板.md`

每个子代理需要：
- 明确的应用场景描述
- 针对特定项目类型的专业指导
- 结合相应模板的最佳实践
- 符合现有 `rust-programmer.md` 的格式规范（主要是指开头部分）
