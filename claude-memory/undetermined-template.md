# 概述

本文档用于描述待定内容的规范模板

# 模板

```
# ✅ <待定项编号>.<可能的目的编号> 建议采纳 - xx % ：

- 原文：（<原文件相对于项目根目录的路径>， <原文所在行数>）

> 原文内容

- 优化后的内容：

> 优化后的内容

- 权衡点：
  - 加分项（建议采纳 + xx %）：……
  - 加分项（建议采纳 + xx %）：……
  - 减分项（建议去除 - xx %）：……
  - 减分项（建议去除 - xx %）：……
  - ……
```

# 示例

>
