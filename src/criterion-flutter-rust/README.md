# Flutter + Rust 协同守则套件

本目录聚焦 Flutter 与 Rust 协同开发场景，涵盖本地 FFI、flutter_rust_bridge、WASM、Tauri 等多种集成方式。文档结构：

- `规范守则_Rust计算协同.md`：对比多种集成方案，给出架构原则、性能与部署建议。
- `项目创建与维护指导.md`：提供从环境搭建、模块划分、CI/CD 到调试部署的完整流程，含不同方案的操作步骤与注意事项。

建议工作流：
1. 根据目标平台与性能需求先查阅“规范守则”，选择适合的Rust集成路径（可组合多方案）。
2. 参考“项目创建与维护指导”执行初始化、模块化、接口定义、测试与发布。
3. 若新增或调整方案，请同步更新本README与CHANGELOG。
