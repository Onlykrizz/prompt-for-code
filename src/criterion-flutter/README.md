# Flutter 项目规范守则套件

本目录针对Flutter跨平台生态的多种交付形态，提供结构化守则模板。文档结合Flutter官方最佳实践、社区工程经验与性能指南，帮助团队在移动、桌面、Web及插件开发中快速达成一致标准。

## 内容总览
- `规范守则_移动与平板应用.md`
- `规范守则_桌面与Web.md`
- `规范守则_插件与平台通道.md`
- `规范守则_企业与可观测性.md`
- `规范守则_高性能与图形动画.md`

## 使用建议
1. 启动新项目时根据目标平台选择对应守则，并结合官方文档执行“核心原则”章节中的必做项。
2. 将文档中的项目结构、CI/CD流程与监控方案融入团队模板或脚手架。
3. 守则列出的参考资料包含Flutter官方站、Dart/Flutter开发手册及社区经验，建议定期复查更新。
4. 如增加新的方向或扩展内容，请同步更新本README和CHANGELOG，保持多语言守则体系一致。
